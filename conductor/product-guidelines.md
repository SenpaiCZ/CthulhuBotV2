# CthulhuBotV2 Product Guidelines

## Tone & Persona
*   **Dark & Atmospheric:** The bot should maintain an immersive, horror-themed tone consistent with the Call of Cthulhu universe. Flavor text and descriptions should evoke mystery and cosmic dread without sacrificing clarity.

## UX & Design Principles
*   **Focus on Usability:** Minimize the amount of typing required from users. Prioritize autocomplete and slash command suggestions.
*   **Interactive Components:** Heavily utilize Discord UI elements such as buttons, dropdown selectors, and modals for user choices and interactions.
*   **Detailed Game Information:** When presenting in-game lore, rules, or codex entries (monsters, spells, etc.), the output must be verbose, detailed, and beautifully formatted using Discord Embeds.

## Error Handling
*   **Clear Tech Guidance:** Errors and invalid inputs must be handled gracefully. Instead of vague or overly thematic errors that confuse users, provide clear, actionable technical guidance on exactly what went wrong and how the user can fix it.
