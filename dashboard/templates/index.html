Commands Reference

  • /newinvestigator — Start character creation wizard
  • /mycharacter — View your character sheet
  • /stat — View or edit specific stats
  • /rename / /renameskill — Rename character or skill
  • /addbackstory / /updatebackstory / /removebackstory — Manage backstory
  • /generatebackstory — Generate random backstory
  • /retire / /unretire — Retire or restore a character
  • /printcharacter — Export character sheet as image
  • /deleteinvestigator — Delete character

  • /roll — Skill check with bonus/penalty dice, luck spend, push
  • /showluck — Show server luck threshold
  • /session start / /session show / /session wipe — Session management

  • /play — Play a song or playlist from YouTube
  • /pause / /resume — Pause or resume playback
  • /skip — Skip current track
  • /stop — Stop and disconnect
  • /volume — Set volume (0–100)
  • /loop — Toggle loop
  • /queue — View queue
  • /playnext — Add song to front of queue
  • /remove / /clear — Remove songs from queue
  • /nowplaying — Show current track

  • /monster — Monster stats & lore
  • /deity — Deity description & cult info
  • /spell — Spell details
  • /weapon — Firearm/weapon stats
  • /occupationinfo — Occupation lookup
  • /skill — Skill lookup
  • /randomnpc — Generate NPC
  • /randomname — Generate 1920s name
  • /archetype / /talent — Pulp content
  • /phobia / /mania — Madness tables
  • /poison / /invention / /year — Reference
  • /loot / /macguffin — Loot & props

  • /karma / /leaderboard — Karma system
  • /reactionrole — Reaction roles
  • /poll / /remind — Polls & reminders
  • /purge — Bulk delete messages
  • /autoroom — Auto-room management
  • /rss / /rsssetup — RSS feeds
  • /backup / /updatebot — Maintenance
  • !sync — Sync slash commands
